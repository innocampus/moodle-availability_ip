# moodle-availability_ip

Moodle availability plugin to allow restricting activities and sections by IP address.

## Requirements

This plugin requires at least Moodle `4.5`.

## Installation

Install the plugin by copying the code to `availability/condition/ip`.

Example:

```shell
git clone \
    <REPOSITORY_BASE_URL>/moodle-availability_ip.git \
    availability/condition/ip
```
